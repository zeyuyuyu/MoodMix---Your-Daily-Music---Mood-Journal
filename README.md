# MoodMix

A beautifully designed web app that helps users track their daily mood through music. Users select their current mood and receive a perfectly curated playlist. The app creates stunning visualizations of their mood patterns over time, synchronized with their music history.

Key Features:
- Interactive mood selector with beautiful animations
- Spotify-style music player interface
- Personal mood tracking calendar
- Shareable mood & music cards
- Progressive Web App support

Tech Stack:
- Vue.js 3 for reactive UI
- Anime.js for smooth animations
- Chart.js for mood visualizations
- Local Storage for offline support
